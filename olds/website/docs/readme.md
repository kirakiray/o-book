# 说明

**o-book** 是一个非常简单的书籍网站生成框架；可以把文档整理成电子书籍发布，便于查阅；

基于 [ofajs](https://ofajs.com/) 开发，不需要安装nodejs环境，不需要学习命令行；

只需要静态服务器，创建html文件，引用相关组件即可；

当前网站也是使用 **o-book** 制作的；

它使用 **Markdown** 来编排书本；

是 **gitbook** 的替代品；（如果你没听过可以忽略这段）
